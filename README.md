# Photon Energia

Site institucional da Photon Energia, empresa especializada em soluções de energia solar e sustentabilidade. O projeto apresenta os serviços oferecidos, projetos realizados e informações sobre migração para o mercado livre de energia.

## Sobre o Projeto

O Photon Energia é uma aplicação web moderna desenvolvida com Next.js que serve como vitrine digital para a empresa, apresentando:

- Soluções em energia solar para residências, indústrias e agronegócio
- Portfólio de projetos realizados com galeria de imagens
- Informações sobre migração para o mercado livre de energia
- Casos de sucesso destacados
- Seção de perguntas frequentes (FAQ)
- Informações sobre a empresa e seus parceiros


## Funcionalidades Principais

### Hero
Carrossel de banners com efeitos de transição e animações, apresentando os principais serviços:
- Energia Solar
- Bombeamento Solar
- Treinamentos
- Sustentabilidade
- Eficiência Energética

### Seção de Soluções
Apresenta três categorias principais:
- **Residencial**: Soluções para residências
- **Industrial**: Soluções para empresas
- **Agronegócio**: Soluções para o setor rural

### Galeria de Projetos
Exibe projetos realizados organizados por categoria:
- Projetos industriais
- Projetos residenciais
- Projetos rurais

Cada projeto possui uma galeria de imagens com visualização em modal e navegação entre imagens.

### Caso de Sucesso - Hospital
Seção dedicada ao projeto do Hospital São Vicente de Paula em Abaeté, MG, com detalhes técnicos e galeria de imagens.

### Mercado Livre de Energia
Informações sobre os serviços oferecidos para migração ao mercado livre de energia, incluindo:
- Migração simplificada
- Suporte especializado
- Atualização contínua
- Representação na CCEE
- Gestão inteligente de energia

### FAQ
Seção de perguntas frequentes com respostas sobre energia solar, manutenção, retorno do investimento e funcionamento em dias nublados.

### Sobre a Photon
Apresentação da empresa com vídeo institucional.

### Parceiros
Carrossel com logos dos parceiros da empresa.

## Instalação e Execução

### Pré-requisitos
- Node.js 18+ 
- npm, yarn, pnpm ou bun

### Instalação

1. Clone o repositório:
```bash
git clone <url-do-repositorio>
cd photon-energia
```

2. Instale as dependências:
```bash
npm install
# ou
yarn install
# ou
pnpm install
```

### Execução em Desenvolvimento

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

O servidor de desenvolvimento estará disponível em `http://localhost:3000`.

### Build para Produção

```bash
npm run build
npm start
```
